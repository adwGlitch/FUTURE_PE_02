## Project Highlights

This content pack contains all the creative and analytical assets required to deploy a direct-response video campaign:

*   **20 Scroll-Stopping Hooks:** Classified into Question, Problem, Curiosity, Emotional, and Statistic-based openers to maximize early retention.
*   **10 UGC Ad Scripts:** Conversational 30–60 second video scripts focusing on authentic creator-style delivery.
*   **40 Short-form Video Scripts:** Custom-tailored vertical video drafts (10 Instagram Reels, 10 YouTube Shorts, 10 Facebook Reels, and 10 TikTok Shorts).
*   **15 Marketing Captions:** High-engagement copy with strategic emoji usage and CTA prompts.
*   **20 CTAs:** Actionable and benefit-driven buttons and screen overlays to drive sign-ups.
*   **Prompt Engineering Documentation:** Systematic breakdowns of prompt design, persona setup, and negative constraints.
*   **Prompt Evaluation:** A comparison matrix tracking the progression from V1 to V2 to the Final system prompt.
*   **GitHub Documentation:** Clear documentation files detailing assets, metrics, and output logs.

## Introduction

Managing daily schedules is increasingly complex. Traditional productivity tools often introduce friction rather than solving it, leading to cognitive fatigue and dropped deadlines. 

Reminderr solves this by turning natural conversational inputs into action items. This repository contains the full UGC ad copy package generated using structured prompting techniques. The goal was to build direct-response assets that explain the product naturally without falling back on cliché marketing terminology.

## Prompt Engineering Techniques

To generate natural language copy and structured directories, several prompt engineering methodologies were implemented:

1.  **Persona Prompting:** Instructing the model to act as a *Performance Marketer*, *UGC Creator*, and *Prompt Engineer* to combine conversion focus with natural syntax and directory styling.
2.  **Negative Constraints:** Restricting the model from using robotic AI words like *"revolutionary"*, *"game-changing"*, and *"cutting-edge"*, forcing it to write conversationally.
3.  **Structured Templating:** Injecting exact Markdown templates inside the prompt to dictate headings, columns, and layout before generation occurred.
4.  **Zero-Shot Prompting:** Generating high-quality original concepts and visual cues from structured directives without sample data.
