## 🔍 Project Overview
**Reminderr** is a modern, friendly productivity assistant that turns natural text or voice conversations into context-aware reminders. 

This repository contains a complete ad content pack designed to increase app sign-ups. The campaign focuses on shifting user perception from **stress/cognitive overload** (manual calendar setups, messy sticky notes) to **calm/control** (frictionless natural language scheduling).

## 🧠 Prompt Engineering Techniques
To generate natural language copy and structured directories, several prompt engineering methodologies were implemented:
1.  **Persona Prompting:** Instructing the model to act as a *Performance Marketer*, *UGC Creator*, and *Prompt Engineer* to combine conversion focus with natural syntax and directory styling.
2.  **Negative Constraints:** Restricting the model from using robotic AI words like *"revolutionary"*, *"game-changing"*, and *"cutting-edge"*, forcing it to write conversationally.
3.  **Structured Templating:** Injecting exact Markdown templates inside the prompt to dictate headings, columns, and layout before generation occurred.
4.  **Zero-Shot Prompting:** Generating high-quality original concepts and visual cues from structured directives without sample datasets.
