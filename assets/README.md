# Campaign Assets Directory

This directory is structured to hold the visual elements, design templates, and reference materials for the Reminderr UGC Ad Campaign.

## Directory Structure

*   `screenshots/` - Place active web application UI screenshots showing the Reminderr dashboard, natural language input box, and notification pop-ups.
*   `mockups/` - Place phone and laptop mockups showing vertical video ad placements (Reels, TikTok) and the app's lockscreen widget in action.
*   `references/` - Place visual style guides, typography notes, and color palettes.

## Design Reference Guidelines

To maintain brand consistency across all ad variants, ensure that visual assets follow these guidelines:

### 1. Typography
*   **Primary Font:** Outfit (Google Fonts) - used for bold hook headings and titles.
*   **Body Font:** Inter (Google Fonts) - used for clean UI text and subtitles.

### 2. Brand Color Palette
*   **Primary Dark (Background):** `#090D16` (Deep, sleek navy-black)
*   **Secondary Dark (Widgets):** `#161B26` (Clean dark blue-grey)
*   **Accent Teal (Highlights):** `#0D9488` (Modern teal)
*   **Accent Coral (Important):** `#F43F5E` (Warm coral for overdue indicators)
*   **Text Light:** `#F3F4F6` (Off-white for premium readability)

### 3. Visual Aesthetic
*   **Style:** Minimalist dark mode, glassmorphism card layouts (using HSL colors with alpha transparency), smooth gradients, and high-quality workspace backgrounds.
*   **Pacing:** Cuts every 1.5 to 2.5 seconds to keep engagement rates high.
